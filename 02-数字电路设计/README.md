# 02 - 数字电路设计

> 数字电路是芯片的"语言"。理解RTL设计，是理解芯片功能实现的基础。

## 📚 本章节学习目标

- [ ] 掌握组合逻辑和时序逻辑设计
- [ ] 理解Verilog/SystemVerilog基本语法
- [ ] 理解时序约束和时钟域
- [ ] 能阅读简单RTL代码

## 🎯 核心知识点

### 2.1 组合逻辑
- 基本逻辑门（与、或、非、异或）
- 多路选择器、译码器、编码器
- 加法器、乘法器设计
- 布尔代数与卡诺图
- **PM视角**: 理解逻辑复杂度与面积/延迟的关系

### 2.2 时序逻辑
- 触发器（DFF、JK、T）
- 寄存器与锁存器
- 有限状态机（FSM）
- 同步 vs 异步设计
- **PM视角**: 时序是芯片设计的核心约束

### 2.3 Verilog/SystemVerilog
- 模块定义与实例化
- 数据类型与运算符
- 行为级与结构级建模
- Testbench编写
- **PM视角**: 能读懂RTL，能与工程师有效沟通

### 2.4 时钟与复位
- 时钟树与时钟域
- 跨时钟域（CDC）处理
- 同步复位 vs 异步复位
- **PM视角**: 时钟设计直接影响芯片功耗和面积

### 2.5 FPGA基础
- FPGA架构（LUT、FF、BRAM、DSP）
- FPGA vs ASIC的区别
- **PM视角**: FPGA是原型验证和中小批量的选择
