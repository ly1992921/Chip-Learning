# 01 - 半导体物理基础

> 理解半导体器件的物理原理，是理解芯片一切的基础。
> 作为PM，你需要理解器件特性如何影响芯片性能和成本。

## 📚 本章节学习目标

- [ ] 理解晶格结构和能带理论
- [ ] 掌握PN结和MOS结构的工作原理
- [ ] 理解CMOS反相器和基本逻辑门
- [ ] 了解存储器的基本分类和原理

## 🎯 核心知识点

### 1.1 固体物理基础
- 晶体结构与晶格
- 能带理论（导带、价带、带隙）
- 载流子（电子、空穴）
- 掺杂（N型、P型半导体）
- **PM视角**: 带隙决定材料特性（Si vs GaN vs SiC）

### 1.2 PN结与二极管
- 耗尽区与内建电势
- 正向偏置与反向偏置
- I-V特性曲线
- **PM视角**: 理解功率器件与逻辑器件的区别

### 1.3 MOS结构与晶体管
- MOS电容结构
- MOSFET工作原理
- 阈值电压、跨导、亚阈值斜率
- I-V特性与开关特性
- **PM视角**: 理解晶体管是芯片的"乐高积木"

### 1.4 CMOS技术
- CMOS反相器
- 功耗来源（动态、静态、短路）
- CMOS逻辑门设计
- **PM视角**: CMOS是数字芯片的基础，功耗是关键指标

### 1.5 存储器基础
- SRAM: 6T单元结构
- DRAM: 1T1C单元结构
- Flash: 浮栅晶体管
- **PM视角**: 不同存储器的速度/密度/成本权衡
