# 05 - 芯片制造工艺

> 制造工艺决定了芯片的物理极限。理解Fab流程，是理解芯片成本和性能的基础。

## 📚 本章节学习目标

- [ ] 理解晶圆制造的完整流程
- [ ] 掌握光刻技术的基本原理
- [ ] 了解先进制程节点（FinFET、GAA）
- [ ] 理解良率和可靠性管理

## 🎯 核心知识点

### 5.1 制造流程概览
- 晶圆制备（拉晶、切片、抛光）
- 氧化、光刻、刻蚀、沉积
- 离子注入、退火
- 金属化与CMP
- **PM视角**: 制造流程的复杂度和成本

### 5.2 光刻技术
- 光源演进（g-line → i-line → DUV → EUV）
- 分辨率极限与瑞利判据
- 多重图案化（SADP/SAQP）
- **PM视角**: 光刻是制程推进的最大瓶颈

### 5.3 先进制程
- FinFET: 3D晶体管结构
- GAA (Gate-All-Around)
- 3nm/2nm技术挑战
- 新材料（High-k、Ge、2D材料）
- **PM视角**: 制程节点对性能/功耗的影响

### 5.4 良率与可靠性
- 缺陷类型与来源
- 良率模型
- 老化与可靠性测试
- **PM视角**: 良率直接决定芯片成本

### 5.5 封装技术
- 传统封装（DIP/QFP/BGA）
- 先进封装（WLCSP、Fan-out）
- **PM视角**: 封装也是芯片价值链的重要环节
