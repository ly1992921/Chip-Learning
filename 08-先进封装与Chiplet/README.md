# 08 - 先进封装与Chiplet

> 先进封装正在成为延续摩尔定律的关键路径。Chiplet技术正在重塑芯片设计范式。

## 📚 本章节学习目标

- [ ] 理解先进封装技术（CoWoS、InFO、EMIB）
- [ ] 掌握HBM内存架构
- [ ] 理解Chiplet设计理念和UCIe标准
- [ ] 了解先进封装的市场格局

## 🎯 核心知识点

### 8.1 先进封装技术
- 2.5D封装（CoWoS、EMIB）
- 3D封装（SoIC、Foveros）
- Fan-out封装（InFO）
- **PM视角**: 封装不再是附属，而是核心技术

### 8.2 HBM (High Bandwidth Memory)
- HBM架构（堆叠DRAM + TSV）
- HBM2/HBM2E/HBM3演进
- 带宽与容量
- **PM视角**: HBM是AI芯片的"粮草"

### 8.3 Chiplet设计
- 模块化设计理念
- Die-to-Die互联
- UCIe标准
- **PM视角**: Chiplet降低设计成本，加速迭代

### 8.4 互联技术
- 片内互联（NoC、AXI）
- 片间互联（UCIe、CXL）
- 光互联
- **PM视角**: 互联带宽决定系统扩展性
