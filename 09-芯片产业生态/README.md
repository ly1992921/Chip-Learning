# 09 - 芯片产业生态

> 芯片是全球分工最细、链条最长的产业之一。理解产业生态，是芯片PM的战略基础。

## 📚 本章节学习目标

- [ ] 理解IDM/Fabless/Foundry三种模式
- [ ] 掌握全球芯片产业链格局
- [ ] 了解主要芯片厂商的产品和战略
- [ ] 理解地缘政治对芯片产业的影响

## 🎯 核心知识点

### 9.1 产业模式
- IDM (Integrated Device Manufacturer): Intel, Samsung
- Fabless: NVIDIA, AMD, Qualcomm, 华为海思
- Foundry: TSMC, Samsung, GlobalFoundries, SMIC
- OSAT: ASE, Amkor, JCET
- **PM视角**: 不同模式对PM工作的要求不同

### 9.2 全球产业链
- 设计: 美国、中国、欧洲
- 制造: 中国台湾、韩国、中国大陆
- 设备: 荷兰(ASML)、美国(Applied Materials)、日本(Tokyo Electron)
- 材料: 日本、德国、美国
- **PM视角**: 供应链安全是战略问题

### 9.3 主要厂商分析
- NVIDIA: GPU/AI芯片霸主
- AMD: CPU+GPU双轮驱动
- Intel: IDM转型
- Qualcomm: 移动芯片
- 华为海思: 国产替代代表
- **PM视角**: 每家公司的战略选择和产品路线

### 9.4 地缘政治
- 中美芯片博弈
- 出口管制与制裁
- CHIPS法案与各国补贴
- **PM视角**: 地缘政治影响产品规划和市场策略

### 9.5 投资与并购
- 芯片行业投资逻辑
- 经典并购案例
- 创业机会
- **PM视角**: 理解行业资本动向
